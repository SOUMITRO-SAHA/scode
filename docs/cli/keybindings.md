# Keyboard Shortcuts

## Chat Area

| Key                     | Action                   |
| ----------------------- | ------------------------ |
| `Ctrl+@` / `Ctrl+Enter` | Send message             |
| `Ctrl+L`                | Clear chat               |
| `PageUp` / `PageDown`   | Scroll chat history      |
| `↑` / `↓`               | Navigate message history |

## Sidebar

| Key                 | Action                            |
| ------------------- | --------------------------------- |
| `Ctrl+B`            | Toggle session sidebar            |
| `Tab` / `Shift+Tab` | Navigate between sidebar and chat |

## Composer / Input

| Key          | Action                         |
| ------------ | ------------------------------ |
| `Ctrl+Space` | Trigger autocomplete           |
| `Tab`        | Accept autocomplete suggestion |
| `Esc`        | Dismiss autocomplete           |
| `Ctrl+U`     | Clear input line               |
| `Ctrl+W`     | Delete word backward           |

## Command Palette

| Key     | Action                   |
| ------- | ------------------------ |
| `/`     | Open command palette     |
| `Esc`   | Close command palette    |
| `Enter` | Execute selected command |

## Navigation

| Key      | Action                           |
| -------- | -------------------------------- |
| `Ctrl+C` | Cancel streaming response / quit |
| `Ctrl+D` | Exit scode                       |
| `Ctrl+Z` | Suspend (Unix)                   |

## Debugging

| Key                      | Action                           |
| ------------------------ | -------------------------------- |
| `Ctrl+T`                 | Toggle thinking panel visibility |
| `Ctrl+D` (during stream) | Toggle debug information         |
